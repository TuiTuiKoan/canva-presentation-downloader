This small solution helps people download the full presentation slide as a PDF with a CANVA link shared by the original author. All hyperlinks are accessible but kept as additional information based on the technical constraints.

You can direct download install files(.exe or macos executional file) from the "dist" folder.

You might encounter some warning messages when you install the file. These warnings are standard for independent open-source software that hasn't been notarized by Microsoft or Apple.

To prevent you from thinking the program is broken, please take note of the following:

**For Windows Users (app-win.exe)**

Security Alert: When double-clicking the file for the first time, Windows Defender or SmartScreen will likely display a blue "Windows protected your PC" warning. This occurs because the .exe file does not have a paid digital signature from Microsoft.

The Solution: 
- Please click "More info" on the warning screen. 
- Then click "Run anyway" to launch the terminal interface.

**For Mac Users (app-macos)**

Unidentified Developer: Mac’s Gatekeeper security is very strict. If you double-click the file directly, the system may block it, stating it is from an unidentified developer.

The Solution: 
- Locate app-macos in Finder.
- Right-click (or Control-click) the file and select "Open".
- A warning dialog will appear. Click "Open" again in that dialog to manually authorize and run the application.

這個小工具可以幫助使用者透過原作者分享的CANVA連結，將完整的簡報檔案下載為 PDF。受限於技術限制，投影片中的所有超連結皆可點擊，但會以補充資訊的形式呈現。

您可以直接從 「dist」 資料夾下載安裝檔（包含 Windows 的 .exe 檔或 macOS 的執行檔）。

安裝過程中，您可能會遇到安全警告訊息。這些警告對於未經過微軟（Microsoft）或蘋果（Apple）付費數位簽章認證的獨立開源軟體來說，屬於正常現象。
  
為了避免您以為程式壞掉，請留意以下狀況：

**給 Windows 使用者 (app-win.exe)：**

防護提示：第一次雙擊點開時，Windows Defender 或 SmartScreen 很高機率會彈出「Windows 已保護您的電腦」的藍色警告畫面（因為這個 .exe 沒有花錢向微軟購買數位簽章）。

解法：
- 請點擊警告畫面上的「其他資訊」
- 然後按下「仍要執行」即可順利開啟終端機介面。

**給 Mac 使用者 (app-macos)：**

未受信任的開發者：Mac 的 Gatekeeper 安全機制非常嚴格。如果直接點擊，系統可能會說這是不明開發者製作的應用程式而拒絕開啟。

解法：
- 在 Finder 中，對著 app-macos 點擊右鍵，然後選擇「打開」。
- 系統會跳出警告框，這時再點擊一次警告框裡面的「打開」，就能強制授權執行了。
